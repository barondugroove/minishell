## 🔩 Makefile

### This project use `Make`, here are the commands to compile it

**1. 🗜️Compiling the project :**

```shell
$ make
```

**2. 🧹Cleaning all binary (.o) :**

```shell
$ make clean
```

**2. 🪦Cleaning all binary (.o) and executable file :**

```shell
$ make fclean
```
